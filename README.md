# quiz-scorer

A simple slackbot to post new quizzes and keep track of users' scores.

## Installing

Requirements:
* Python 3.6+

1. Clone the repo
2. Install the requirements

```
pip install -r requirements.txt
```

3. Run it

```
$ python app.py
```

## Environment variables

* `SLACK_BOT_TOKEN`: get this from slack
* `PROXY`: proxy
